# Assets

Shared visual and audio support assets copied from the live workspace.

These are included because cover assembly and video rendering reference them directly. Generated daily media is intentionally kept out of this folder.

## Lumi

- `lumi/`: stable Lumi character references for imagegen cover prompts and video host identity checks.
- `lumi-avatar.png`, `lumi-cover-ref.png`, and `lumi-dance.gif`: legacy/runtime Lumi assets still referenced by existing rendering paths.
