# Lumi Character References

This folder contains the stable Lumi identity references used by cover and video
generation prompts.

These files are source references, not daily generated outputs.

## Files

- `lumi-reference-fullbody.png`: main full-body identity reference. Use this for
  Lumi's proportions, outfit, pink hair, black bow, student-secretary energy, and
  readable chest badge.
- `reference/lumi-approved-reference-2026-04-29.png`: approved cover crop from
  the 2026-04-29 iteration. Use this for facial expression, bow placement, badge
  treatment, and cover-scale personality.

## Prompt Contract

- Keep Lumi close to the approved pink-haired, black-bow student-secretary look.
- Keep the small `Lumi` chest badge readable when Lumi is large enough.
- In cover thumbnails, Lumi should stay in a lower corner and should not become
  the dominant news subject unless the brief explicitly asks for a character-led
  cover.
- Daily news story images should still come from official or web sources. These
  Lumi references are for the host character only.
